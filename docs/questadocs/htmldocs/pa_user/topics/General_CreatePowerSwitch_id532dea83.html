Usage Notes

  • If the boolean expression of an ON or PARTIAL_ON state evaluates to X or Z, and the corresponding state of the input supply port is FULL_ON or PARTIAL_ON, then the output of the switch is set to UNDETERMINED state.

  • If you specify a boolean_function in the -ack_port option, the tool drives the result of the boolean_function on the ack port with a delay of -ack_delay time units after a control port transition.

  • The -instances option is introduced in UPF 2.1, but you can enable the option for UPF 2.0 by using the following argument: 

    vopt ‑pa_upfextensions=relaxedsyntax

  • The create_power_switch command supports multiple ON states. The -output_voltage option associates every ON state with a voltage value.

    create_power_switch
...  
	[-output_voltage {state_name voltage_value}]* 
...

create_power_switch
