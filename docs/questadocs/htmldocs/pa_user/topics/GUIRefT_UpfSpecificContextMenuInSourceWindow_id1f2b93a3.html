UPF-Specific Context Menu in the Source Window

To access: Right-click in the Source window

Use this context menu to perform Power Aware actions for a selected signal.

Objects

Menu Item

Description

UPF > View

  • Power Domain

  • Isolation Strategy

  • Level Shifter Strategy

  • UPF Supply Connection

Opens the corresponding UPF file, highlighting the line that defines the behavior.

Prints information about the UPF behavior to the Transcript window.

UPF > Add To Wave

  • Power Domain

  • Isolation Cell

  • Level Shifter Cell

  • UPF Supply Connection

Adds Power Aware information to the Wave window.
