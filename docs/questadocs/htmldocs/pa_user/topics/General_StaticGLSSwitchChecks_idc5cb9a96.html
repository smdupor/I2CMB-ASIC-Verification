Static GLS Switch Checks

Static GLS switch checks report any missing or inconsistent switch cells in your gate-level design.

When you enable static GLS checks (using vopt -pa_glschecks=s), the tool performs the static GLS switch checks listed in the following table. All static GLS switch check results are written to the report file, report.static.txt.

Table 1. Static GLS Switch Checks

Mnemonic

Description

SWTCH_CTRL_NOT_ALWYS_ON

Reports switch control signals that are not generated from the always-on region of the design.

SWTCH_CTRL_UNCONNECTED

Reports switch control signals that are unconnected or floating in the design.

SWTCH_CTRL_CONST_DRIVER

Reports switch control signals that are driven by constants in the design.
