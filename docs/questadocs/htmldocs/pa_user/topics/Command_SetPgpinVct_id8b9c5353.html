set_pgpin_vct

Specifies the VCT to be used for pins of any particular pg_type attribute.

Syntax

set_pgpin_vct <pg_type> -vct <vct_name>

Description

Specify the set_pgpin_vct command in either a UPF or Tcl file, and include the file in the vopt run using the -pa_upf or -pa_tclfile argument.

Arguments

<pg_type>

Identifies the value of a pg_type attribute, such as primary_power, primary_ground, backup_power, backup_ground, nwell, pwell, deepnwell, deeppwell, internal_power, or internal_ground.

-vct <vct_name>

Identifies the VCT.

Examples

set_pgpin_vct primary_power -vct UPF2VHDL_SL
