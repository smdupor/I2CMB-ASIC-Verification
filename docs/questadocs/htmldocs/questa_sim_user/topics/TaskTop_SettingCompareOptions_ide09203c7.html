Setting Compare Options

There are a few global options that you can set for a comparison.

Procedure

  1. Select Tools > Waveform Compare > Options to open the Comparison Options dialog box (Figure 1).
  2. The General Options tab in this dialog allow you to set the maximum number of differences allowed before the comparison terminates, specify signal value matching rules, and save or reset the defaults.
