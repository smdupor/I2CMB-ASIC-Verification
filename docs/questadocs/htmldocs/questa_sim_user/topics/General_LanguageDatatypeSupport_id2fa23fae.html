Language and Datatype Support

Questa SIM code coverage supports VHDL and Verilog/SystemVerilog language constructs.

Code Coverage collects data separately for Verilog tasks and functions and VHDL subprograms, collectively referred to as “subprograms.” It stores data for subprograms in the UCDB as separate regions, and reports on that data separately when you use the coverage report or vcover report commands. And it displays all subprograms - Verilog tasks and functions as well as VHDL subprograms - in the Structure window of the GUI. (See Code Coverage in the Graphic Interface.)

Code coverage does not work on SystemC design units.

Statement and Branch coverage have no limitations on support, but sometimes optimizations can make it appear that some statements or branches are not considered for coverage. See “Notes on Coverage and Optimization” for more details.
