Requirements for Parallel Merge

There is one requirement for the successful set-up and use of the parallel merge process:

  • License feature ‘qvrm’ — Parallel merge requires the running of VRM, which is a standard part of the Questa install but requires a license feature called 'qvrm'. The feature is available with Questa SV and AFV, Questa Prime and Ultra or as an option to Questa Core called Questa Verification Management.
