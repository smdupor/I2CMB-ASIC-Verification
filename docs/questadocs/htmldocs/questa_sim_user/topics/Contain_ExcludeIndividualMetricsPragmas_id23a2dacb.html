Exclude Individual Metrics with Pragmas

Questa SIM supports the use of source code pragmas to selectively turn coverage off and on for individual code coverage metrics.

Pragmas enable you to turn statement, branch, condition, expression, toggle, and FSM coverage on and off independently.
