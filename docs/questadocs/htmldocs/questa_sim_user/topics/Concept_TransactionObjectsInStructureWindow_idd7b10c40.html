Transaction Objects in the Structure Window

The Structure windows allow you to navigate through the regions in the design, which is similar to using the env command in command line mode.

Transaction objects look much like signals or nets in the design hierarchy of the Structure window. When you navigate to a region containing a stream, the stream is visible in the Objects window.
