Enabling Sparse Memories

You can enable sparse memories manually by inserting attributes or meta-comments in your code.
Note:

QIS is the preferred method to use when optimizing designs with vopt for debug/visibility purposes. Refer to“Questa Dumping and Visibility” for complete details.

You can also enable sparse memories automatically by setting the SparseMemThreshold variable in the modelsim.ini file
