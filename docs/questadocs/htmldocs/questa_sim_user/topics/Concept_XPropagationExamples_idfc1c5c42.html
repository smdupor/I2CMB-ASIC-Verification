X Propagation Examples

Examples of X propagation can be divided into two categories; RTL x-optimism examples and RTL x-pessimism examples.

You can find additional examples within the install directory at: 

<install_dir>/examaples/xprop/

RTL X-Pessimism Examples

Example 3. Array-index Pessimism
Reg [1:0]arr;
q = arr[idx];

If 'idx' goes x, then RTL simulation assigns 'x' value to q irrespective of values stored in different bits of the vector 'arr'. However in silicon, if all the bits of 'arr' are same then q takes a valid value.
