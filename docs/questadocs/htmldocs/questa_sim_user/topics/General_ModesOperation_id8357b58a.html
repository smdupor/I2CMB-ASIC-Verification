Modes of Operation

The parallel merge process can be run in three main modes of operation.

These modes are as shown in Table 1.

Table 1. Modes for Parallel Merge

Option

Mode

-runmode local

Run Locally on a machine with multiple cores available.

-runmode rsh

Run using rsh to remotely execute merges on defined machines.

-runmode lsf | sge | uge | rtda

Run using various types of grid software.

A full list of arguments used for parallel merge is found in vcover parallelmerge.
