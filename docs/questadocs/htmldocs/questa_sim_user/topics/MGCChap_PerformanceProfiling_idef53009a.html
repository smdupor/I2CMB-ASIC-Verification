Performance Profiling

The Questa SIM statistical sampling profiler provides instance specific execution so you can easily identify areas in your simulation where performance can be improved. The profiler can be used at all levels of design simulation—Functional, RTL, and Gate‑Level—and has the potential to save hours of regression test time. In addition, ASIC and FPGA design flows benefit from the use of this tool.
