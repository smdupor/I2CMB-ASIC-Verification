ToggleVHDLRecords

Section [vsim]

This variable controls the inclusion of VHDL records in toggle coverage metrics. By default, VHDL records are included in coverage.

Syntax

ToggleVHDLRecords = {0 | 1}

Arguments

The arguments are described as follows:

  • 0 — Off

  • 1 — (default) On

    You can override this variable by specifying vsim -notogglevhdlrecords.
