Customizing the Column Views in Verification Windows

You can customize the display of columns in the Verification Browser window or Tracker window, and then save these views for later use.

Procedure

  1. Select [Create/Edit/Remove ColumnLayout...] from the pulldown list.

    This displays the Create/Edit/Remove Column Layout dialog box.

  2. Enter a new name in the Layout Name text entry box.
  3. (Optional) Add or modify pre-defined column arrangements from the Create/Edit/Remove Column Layout dialog box by adding columns to or removing them from the Visible Columns box as desired.
  4. Click OK.

Results

After applying your selections, the rearranged columns and custom layouts are saved and appear when you next open that column view in the Verification Browser or Tracker windows.
