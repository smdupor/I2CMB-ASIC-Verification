Basic VHDL Usage

Using a VHDL design with Questa SIM consists of running the vcom, vopt, and vsim commands to compile, optimize, load, and simulate. Note that you need to be familiar with any setup requirements for running these commands, such as using the vlib command to create a design library.

The following steps summarize the basic VHDL usage process:

  1. Compile your VHDL code into one or more libraries using the vcom command. Refer to Compilation of a VHDL Design—the vcom Command for more information.

  2. (Optional) Elaborate and optimize your design using the vopt command. Refer to the chapter Optimizing Designs with vopt for more information.

  3. Load your design with the vsim command. Refer to Simulation of a VHDL Design—the vsim Command.

  4. Simulate the loaded design, then debug as needed.
