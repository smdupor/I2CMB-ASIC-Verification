IgnoreSVAFatal

Section [vsim] [vopt]

This variable instructs Questa SIM to disable SystemVerilog assertion messages for Fatal severity (for vsim command) and suppresses output of elaboration system $fatal tasks (for vsim and vopt commands).

Syntax

IgnoreSVAFatal = 0 | 1}

Arguments

The arguments are described as follows:

  • 0 — (default) Off. SystemVerilog assertion messages enabled.

  • 1 — On. SystemVerilog assertion messages disabled.
