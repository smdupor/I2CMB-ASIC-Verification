CoverREC

Sections [vcom], [vlog]

This variable controls the collection of code coverage for rapid expression and condition coverage statistics. Disabling (0) REC collection converts non-masking conditions in FEC tables to matching input patterns.

Syntax

CoverREC = {0 | 1}

Arguments

The arguments are described as follows:

  • 0 — Off

  • 1 — (default) On

    You can override this variable by specifying vcom, vlog, or vopt ‑nocoverrec.

Description

Refer to Legacy FEC Reporting for more information on the new REC-style reports vs. old style FEC reports.
