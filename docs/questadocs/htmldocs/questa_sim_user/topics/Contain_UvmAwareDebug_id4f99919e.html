UVM-Aware Debug

UVM-Aware Debug provides information at the UVM abstraction level that connects you to the UVM base-class library.

The Questa SIM installation tree includes the latest qualified, pre-compiled version of the Accellera UVM package library. It also contains a pre-compiled DPI library necessary for UVM integration intoQuesta SIM, as well as a UVM debugging package called questa_uvm_pkg that is specific to Questa SIM. These three pieces work together to provide both standard UVM support plus theQuesta SIM capabilities of UVM-Aware debugging.
