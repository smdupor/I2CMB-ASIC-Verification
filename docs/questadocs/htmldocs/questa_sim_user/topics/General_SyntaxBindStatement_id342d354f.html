Syntax of bind Statement

To bind a SystemVerilog assertion module to a VHDL design, the syntax of the bind statement is: 
bind <target_entity/architecture_name> <assertion_module_name> 
	<instance_name> <port connections>

To bind a SystemVerilog assertion module to a SystemC module, the syntax is:

bind <target SystemC module/full hierpath of an instance of a SystemC module> 
	<assertion_module_name> <instance_name> <port connections>

This bind statement creates an instance of the assertion module inside the target VHDL entity/architecture or SystemC module with the specified instance name and port connections. When the target is a VHDL entity, the bind instance is created under the last compiled architecture. Note that the instance being bound cannot contain another bind statement. In addition, a bound instance can make hierarchical reference into the design.
