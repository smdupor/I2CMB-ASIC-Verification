Deleting UCDB Files from the Browser Window

Remove UCDB files from the Verification Browser window.

Procedure

  1. Highlight the test(s) in the Verification Browser window.
  2. Press the Delete key.

Results

A popup appears for you to confirm if you want to delete the selected test(s) from the Verification Browser window and/or from the directory itself.
