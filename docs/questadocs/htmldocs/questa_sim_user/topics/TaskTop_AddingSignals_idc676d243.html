Adding Signals

Add signals for a waveform comparison using the Structure Browser as follows.

Procedure

  1. Select Tools > Waveform Compare > Add > Compare by Signal in the Wave window to open the structure_browser window.
  2. Highlight the signals to be used in the comparison.
  3. Click the OK button.
