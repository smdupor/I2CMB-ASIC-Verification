Quitting C Debug

End a SystemC debugging session from the GUI or from the command line.

  • From the GUI:

    Select Tools > C Debug > Quit C Debug.

  • From the command line, enter the following in the Transcript window:

    cdbg quit

    Note:

    It is recommended that you invoke C Debug once for a given simulation and then quit both C Debug and Questa SIM. Starting and stopping C Debug more than once during a single simulation session may cause problems for gdb.
