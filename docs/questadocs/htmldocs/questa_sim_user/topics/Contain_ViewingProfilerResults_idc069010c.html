Viewing Profiler Results

The profiler provides four views of the collected data: Ranked, Design Units, Call Tree and Structural. All four views are enabled by selecting the corresponding View > Profiling > sub-menu item.
Note:

The Ranked, Design Units, Calltree, and Structural windows, by default, only show performance profile data equal to or greater than 1 percent. You can change this with the Profile Cutoff tool in the profile toolbar group.
