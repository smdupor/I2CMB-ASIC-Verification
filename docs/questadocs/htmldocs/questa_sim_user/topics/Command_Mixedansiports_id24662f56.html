MixedAnsiPorts

Section [vlog]

This variable supports mixed ANSI and non-ANSI port declarations and task/function declarations.

Syntax

MixedAnsiPorts = {0 | 1}

Arguments

The arguments are described as follows:

  • 0 — (default) Off

  • 1 — On

    You can override this variable by specifying vlog -mixedansiports.
