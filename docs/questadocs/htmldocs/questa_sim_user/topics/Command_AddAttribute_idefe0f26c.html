add_attribute

The add_attribute system task adds an attribute to a transaction.

Usage

Verilog

$add_attribute(transaction, value, attribute_name)

VHDL

add_attribute(transaction, value, attribute_name)

Arguments

Name

Type

Description

transaction

Verilog: integer

VHDL: TrHandle

Required. Handle for the transaction to which you are adding the attribute.

value

object

Required.

Verilog — object that is the value to be used for the attribute.

VHDL — constant, variable, signal or generic that is the value to be used for the attribute. Valid types: integer, real, bit, boolean, bit_vector, std_logic, std_logic_vector.

attribute_name

string

Optional for Verilog.

Required for VHDL.

The name of the attribute to be added to the transaction.

Default: The name of the variable used for the value parameter if it can be determined, “anonymous” otherwise.

Return Values

Nothing
