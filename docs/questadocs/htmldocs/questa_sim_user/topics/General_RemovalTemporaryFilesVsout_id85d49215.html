Removal of Temporary Files (VSOUT)

The temporary (temp) file named VSOUT is the communication mechanism between the simulator kernel and the Graphical User Interface.

In normal circumstances, this temp file is deleted when the simulator exits. If Questa SIM crashes, however, you need to delete the temp file manually. If you specify the location of the temp file with TMPDIR, you can locate the file more easily for deletion.
