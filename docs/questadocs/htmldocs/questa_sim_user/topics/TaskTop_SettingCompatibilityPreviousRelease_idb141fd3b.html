Setting Compatibility with a Previous Release

You can specify random sequence generation compatibility with a prior Questa SIM letter release (for the SystemVerilog solver).

Procedure

  1. Use either of the following methods:

    • Set the SolveRev variable in the modelsim.ini file to a previous release.

    • Use the -solverev argument with the vsim command to set a previous release.

  2. The release number you designate must consist of release number and letter, such as 6.6a, but only prior letter releases within same number release are allowed. For example, if you are using version 6.6c, you can specify 6.6b, 6.6a, or 6.6, but cannot specify 6.5f.
    Note:

    These instructions do not apply to the SystemC/SCV solver.
