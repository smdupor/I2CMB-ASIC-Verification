SVCrossNumPrintMissingDefault

Section [vlog]

This variable is used in conjunction with SVCrossNumPrintMissing variable, and overrides the default value of the SystemVerilog covergroup option.cross_num_print_missing (defined to be 0 in the IEEE Std 1800-2009).

Note:

You must recompile the design after changing the setting of this variable.

Syntax

SVCrossNumPrintMissingDefault = <n>

Arguments

The arguments are described as follows:

  • <n> — Any non-negative integer where the default is 0.
