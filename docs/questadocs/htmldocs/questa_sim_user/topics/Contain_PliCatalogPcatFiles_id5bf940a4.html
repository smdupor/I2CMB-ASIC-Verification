PLI Catalog (PCAT) Files

The PLI Catalog encapsulates global or per-task access information for each system task or function, which allows the simulator to preserve visibility while retaining high levels of optimization.
Note:

The syntax of PCAT files is a superset of TAB file syntax, which you may already be using.
