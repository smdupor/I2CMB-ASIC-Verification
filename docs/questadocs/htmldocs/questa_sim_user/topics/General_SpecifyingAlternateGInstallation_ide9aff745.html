Specifying an Alternate g++ Installation

Mentor Graphics recommends using the version of g++ that is shipped with Questa SIM on its various supported platforms.

However, if you want to use your own installation, you can do so by setting the CppInstall variable in the modelsim.ini file to the g++ executable location. For example, if your g++ executable is installed in /u/abc/gcc-4.2.1/bin, then you would set the variable as follows:

CppPath /u/abc/gcc-4.2.1/bin/g++
