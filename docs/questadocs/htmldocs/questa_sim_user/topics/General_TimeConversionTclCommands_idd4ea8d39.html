Time Conversion Tcl Commands

The following table provides Tcl time conversion commands.
Table 1. Tcl Time Conversion Commands

Command

Description

intToTime <intHi32> <intLo32>

converts two 32-bit pieces (high and low order) into a 64-bit quantity (Time in Questa SIM is a 64-bit integer)

RealToTime <real>

converts a <real> number to a 64-bit integer in the current Time Scale

scaleTime <time> <scaleFactor>

returns the value of <time> multiplied by the <scaleFactor> integer
