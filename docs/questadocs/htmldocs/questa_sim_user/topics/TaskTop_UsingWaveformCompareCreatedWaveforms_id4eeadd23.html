Using Waveform Compare with Created Waveforms

The Waveform Compare feature compares two or more waveforms and displays the differences in the Wave window. This feature can be used in tandem with Waveform Editor. The combination is most useful in situations where you know the expected output of a signal and want to compare visually the differences between expected output and simulated output.

Procedure

  1. Create a waveform based on the signal of interest with a drive type of expected output
  2. Add the design signal of interest to the Wave window and then run the design
  3. Start a comparison and use the created waveform as the reference dataset for the comparison. Use the text “Edit” to designate a create waveform as the reference dataset. For example:

    compare start Edit sim
    compare add -wave /test_counter/count
    compare run
