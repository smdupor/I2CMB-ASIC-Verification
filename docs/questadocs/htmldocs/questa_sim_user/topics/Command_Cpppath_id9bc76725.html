CppPath

This variable should point directly to the location of the g++ executable, such as:

Syntax

CppPath = <path>

Arguments

The arguments are described as follows:

  • <path> — The path to the g++ executable.

Description

CppPath = /usr/bin/g+

This variable is not required when running SystemC designs. By default, you should install and use the built-in g++ compiler that comes with Questa SIM.
