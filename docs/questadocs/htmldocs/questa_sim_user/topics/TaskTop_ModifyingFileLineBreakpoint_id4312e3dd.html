Modifying a File-Line Breakpoint

You can easily modify a file-line breakpoints.

Procedure

  1. Select Tools > Breakpoints from the Main menus. This will open the Modify Breakpoints dialog (Figure 1), which displays a list of all breakpoints in the design.
  2. When you select a file-line breakpoint from the list and click the Modify button, the File Breakpoint dialog (Figure 1) opens, allowing you to modify the breakpoint.
