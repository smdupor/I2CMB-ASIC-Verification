Linking Cursors

Cursors within the Wave window can be linked together, allowing you to move two or more cursors together across the simulation timeline. You simply click one of the linked cursors and drag it left or right on the timeline. The other linked cursors will move by the same amount of time.

Procedure

  1. You can link all displayed cursors by right-clicking the time value of any cursor in the timeline, as shown in Figure 1, and selecting Cursor Linking > Link All.
    Figure 1. Cursor Linking Menu

  2. You can link and unlink selected cursors by selecting the time value of any cursor and selecting Cursor Linking > Configure to open the Configure Cursor Links dialog (Figure 2).
    Figure 2. Configure Cursor Links Dialog
