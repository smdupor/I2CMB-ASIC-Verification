Opening the Capacity Window

The Capacity window displays a tabular listing of memory capacity data.

Procedure

Choose View > Capacity from the main menu or enter the view command with “capacity” as the window type:

view capacity

Results

This creates the Capacity window that displays memory data for the current design (Figure 1).
