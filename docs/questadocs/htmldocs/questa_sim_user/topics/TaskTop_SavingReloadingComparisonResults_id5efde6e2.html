Saving and Reloading Comparison Results

To save comparison results for future use, you must save both the comparison setup rules and the comparison differences.

Procedure

  1. To save the rules, select Tools > Waveform Compare > Rules > Save. This file will contain all rules for reproducing the comparison. The default file name is "compare.rul."
  2. To save the differences, select Tools > Waveform Compare > Differences > Save. The default file name is "compare.dif."
  3. To reload the comparison results at a later time, select Tools > Waveform Compare > Reload and specify the rules and difference files.
