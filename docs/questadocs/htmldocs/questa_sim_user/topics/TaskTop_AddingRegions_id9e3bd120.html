Adding Regions

Rather than comparing individual signals, you can also compare entire regions of your design.

Procedure

  1. Select Tools > Waveform Compare > Add > Compare by Region to open the Add Comparison by Region dialog.
  2. Enter the desired region into the Reference Region field or click the Browse button to search for and select the desired region.
  3. Click the “Specify a different name for Test Region” if you want to give the Test Region a different name.
  4. Select from the “Compare Signals of Type” options.
  5. Click the OK button.
