Code Coverage Modes

The Questa SIM default coverage collection for RTL constructs includes a covermode option that enables you to control coverage collection of your constructs to meet coverage closure goals.

A covermode corresponds to a set of coverconstructs for coverage collection. The user interface consists of the vopt command-line option for setting covermodes, and the covermode variable in the vopt section of the modelsim.ini file.
