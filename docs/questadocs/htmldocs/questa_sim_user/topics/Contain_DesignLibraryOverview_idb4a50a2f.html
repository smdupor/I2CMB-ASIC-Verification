Design Library Overview

A design library is a directory or archive that serves as a repository for compiled design units.

The design units contained in a design library consist of VHDL entities, packages, architectures, and configurations; SystemC modules; and Verilog modules and UDPs (user-defined primitives).

Design units are classified in two ways.

  • Primary design units — Entities, package declarations, configuration declarations, modules, SystemC modules, and UDPs. A primary design unit within a given library must have a unique name.

  • Secondary design units — Consist of optimized Verilog modules, architecture bodies, and package bodies. A secondary design unit is associated with a primary design unit. Architectures by the same name can exist if they are associated with different entities or modules.
