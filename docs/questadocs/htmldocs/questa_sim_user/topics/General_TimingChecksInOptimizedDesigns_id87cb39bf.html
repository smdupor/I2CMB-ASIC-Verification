Timing Checks in Optimized Designs

When you run a simulation, Questa SIM performs timing checks whether you optimize the design or not. In either, you will generally see the same results of these checks. However, in a cell where there are both interconnect delays and conditional timing checks, you might see different timing check results.

  • Without vopt — Questa SIM evaluates conditional checks with non-delayed values, complying with the original IEEE Std 1364-1995 specification. You can use the ‑v2k_int_delays argument with vsim to ensure compatibility by forcing the IEEE Std 1364-2005 implementation.

  • With vopt — Questa SIM evaluates conditional checks with delayed values, which complies with the IEEE Std 1364-2005 specification.
