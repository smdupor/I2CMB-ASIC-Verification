Files and Map Libraries

Questa SIM must have access to several specific file types in order to simulate your design.

  • design files (VHDL, Verilog, and/or SystemC), including stimulus for the design.

  • Libraries, both working and resource.

  • The modelsim.ini file (automatically created by the library mapping command).

For detailed information about the files that Questa SIM uses during system startup (including the modelsim.ini file), refer to “System Initialization”.
