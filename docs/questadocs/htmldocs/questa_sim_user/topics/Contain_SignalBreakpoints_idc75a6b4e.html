Signal Breakpoints

Signal breakpoints (“when” conditions) instruct Questa SIM to perform actions when the specified conditions are met. For example, you can break on a signal value or at a specific simulator time. When a breakpoint is hit, a message in the Main window transcript identifies the signal that caused the breakpoint.
