Enabling Fine-Grain Analysis

Fine-grain analysis is enabled by using the -capacity argument with the vsim command.

Procedure

Enter the following at the command line to enable fine-grain analysis:

vsim -capacity

This generates capacity data based on the point of declaration. To show the point of allocation as well as the point of declaration, use the “=line” option with -capacity as follows:

vsim -capacity=line

You can then generate a fine-grained point of allocation (line) based report with the write report command as follows:

write report -capacity -l -line
