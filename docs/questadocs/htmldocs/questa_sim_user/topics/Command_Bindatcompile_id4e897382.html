BindAtCompile

Section [vcom]

This variable instructs Questa SIM to perform VHDL default binding at compile time rather than load time.

Syntax

BindAtCompile = {0 | 1}

Arguments

The arguments are described as follows:

  • 0 — (default) Off

  • 1 — On

    You can override this variable by specifying vcom {‑bindAtCompile | -bindAtLoad}.
