Coverage Exclusions

When code coverage is enabled for an entire design, or for the purposes of debugging a particular segment of the design, you can use coverage exclusions to exclude coverage for individual design units, files, lines of code, objects, and so on.

You can exclude coverage objects with the coverage exclude command, or with source code pragmas. When exclusions are applied, they are saved into the UCDB along with all coverage count data. This enables you to generate reports later in Coverage View mode that match the most recent simulation state.
