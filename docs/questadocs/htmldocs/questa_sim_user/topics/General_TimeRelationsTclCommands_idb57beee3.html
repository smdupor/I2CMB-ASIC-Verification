Time Relations Tcl Commands

The following table provides Tcl time relation commands.
Table 1. Tcl Time Relation Commands

Command

Description

eqTime <time> <time>

evaluates for equal

neqTime <time> <time>

evaluates for not equal

gtTime <time> <time>

evaluates for greater than

gteTime <time> <time>

evaluates for greater than or equal

ltTime <time> <time>

evaluates for less than

lteTime <time> <time>

evaluates for less than or equal

All relation operations return 1 or 0 for true or false respectively and are suitable return values for TCL conditional expressions. For example,

if {[eqTime $Now 1750ns]} {
	...
}
