Using the Comparison Wizard

The simplest method for setting up a comparison is using the Comparison Wizard. The wizard is a series of dialogs that walks you through the process.

Procedure

To start the Wizard, select Tools > Waveform Compare > Comparison Wizard from either the Wave or Main window.
