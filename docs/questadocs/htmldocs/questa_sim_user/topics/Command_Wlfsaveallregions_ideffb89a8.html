WLFSaveAllRegions

Section [vsim]

This variable specifies the regions to save in the WLF file.

Syntax

WLSaveAllRegions = {0 | 1}

Arguments

The arguments are described as follows:

  • 0 — (default) Only save regions containing logged signals.

  • 1 — Save all design hierarchy.
