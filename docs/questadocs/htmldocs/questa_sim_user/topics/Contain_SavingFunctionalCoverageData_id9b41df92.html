Saving Functional Coverage Data

By default, no coverage information is saved into a Unified Coverage DataBase (UCDB) for later post-processing applications — regardless of whether coverage statistics were collected for a simulation — unless you explicitly save it, or if you use vsim -coverstore to save a coverstore.

For instructions on how to automatically save coverage from a simulation, see Coverage Auto-save Coverstore.
