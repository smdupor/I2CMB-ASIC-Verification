PLI Catalog Usage

Use the PLI Catalog File (PCAT file) to control autocompilation of PLI files or as a method of access control for system tasks and functions.
Note:

QIS is the preferred method to use when optimizing designs with vopt for debug/visibility purposes. Refer to “Questa Dumping and Visibility” for complete details.
