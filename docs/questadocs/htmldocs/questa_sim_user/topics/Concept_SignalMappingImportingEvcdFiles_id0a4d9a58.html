Signal Mapping and Importing EVCD Files

When you import a previously saved EVCD file, Questa SIM attempts to map the signals in the EVCD file to the signals in the loaded design by matching signals based on name and width.

If Questa SIM can not map the signals automatically, you can do the mapping yourself by selecting a signal, right-clicking the selected signal, then selecting Map to Design Signal from the popup menu. This opens the Evcd Import dialog.

Figure 1. Evcd Import Dialog

Select a signal from the drop-down arrow and click OK.

Note:

This command works only with extended VCD files created with Questa SIM.
