Transaction Generators

Using Standard C and SystemC types for attributes enable transaction generators without additional preparation with SystemC Verification (SCV).

C/C++ and SystemC type support is limited as described in SCV Limitations. If your design includes user-defined types — such as classes, structures, or enumerations — you must provide SCV extensions so that SCV and the Questa SIM simulator can extract the necessary type and composition information to record the type.

For specific details on providing SCV extensions, refer to the SystemC Verification Standard Specification, Version 1.0e.
