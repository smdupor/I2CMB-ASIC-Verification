Using -assertdebug to Debug with Assertions and Cover Directives

Using the optional ‑assertdebug argument with the vsim command instructs Questa SIM to examine all assertions and display assertion details in the Assertions window.
Note:

QIS is the preferred method to use when optimizing designs with vopt for debug/visibility purposes. Refer to “Questa Dumping and Visibility” for complete details.
