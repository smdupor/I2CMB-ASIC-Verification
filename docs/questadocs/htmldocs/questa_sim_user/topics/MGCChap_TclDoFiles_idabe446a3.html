Tcl and DO Files

Tcl is a scripting language for controlling and extending Questa SIM. Within Questa SIM you can develop implementations from Tcl scripts without the use of C code. Because Tcl is interpreted, development is rapid; you can generate and execute Tcl scripts “on the fly” without stopping to recompile or restart Questa SIM. In addition, if Questa SIM does not provide a command you need, you can use Tcl to create your own commands.
Note:

QIS is the preferred method to use when optimizing designs with vopt for debug/visibility purposes. Refer to “Questa Dumping and Visibility” for complete details.
