ToggleDeglitchPeriod

Section [vsim]

This variable controls the period of toggle deglitching.

Syntax

ToggleDeglitchPeriod = {“n <time_unit>”}

Arguments

The arguments are described as follows:

  • [n] — A string specifying the number of time units. A value of zero ( 0 ) eliminates delta cycle glitches.

  • <time_unit> — (required) fs, ps, ns, us, ms, or sec.

    You must surround n <time_unit> with quotation marks (“ “) when you put a space between “n” and <time_unit>.
