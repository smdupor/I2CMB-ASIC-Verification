TLM and WLM Connections

A Questa Verification IP can be regarded as a protocol interface that has ‘ends’ for connecting a master, slave, clock generator, reset generator, etc, as TLM-connected, WLM-connected, or both.

An interface ‘end’ that is TLM-connected drives the wire-level signals, as a consequence of a transaction being generated within the Questa Verification IP. An interface ‘end’ that is WLM-connected will monitor wire-level signals changes, and attempt to recognize them into transactions.
