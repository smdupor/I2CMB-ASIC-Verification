$psprintf()

The $psprintf() system function behaves like the $sformat() file I/O task, except that the string result is passed back to the user as the function return value for $psprintf(), and is not placed in the first argument as for $sformat(). Thus, you can use $psprintf() where a string is valid. Note that at this time, unlike other system tasks and functions, $psprintf() cannot be overridden by a user-defined system function in the PLI.

Syntax

$psprintf();

Arguments

none
