Expanding and Collapsing Simulation Time

Simulation time may be expanded to view delta time steps or event time steps at a single simulation time or over a range of simulation times. Simulation time may be collapsed to hide delta time steps or event time steps at a single simulation time or over a range of simulation times. You can expand or collapse the simulation time with menu selections, toolbar selections, via commands, or with the mouse cursor.

Do the following:

Menu Selections

Select Wave > Expanded Time when the Wave window is docked, and View > Expanded Time when the Wave window is undocked. You can expand/collapse over the full simulation time range, over a specified time range, or at the time of the active cursor,.

Toolbar Selections

The Debug Toolbar Tab (described in the GUI Reference Manual) includes four buttons for expanding and collapsing simulation time in the Wave window: Expand Full, Expand Cursor, Collapse Full, and Collapse Cursor.

Commands

There are six commands for expanding and collapsing simulation time in the Wave window.

  • wave expand all

  • wave expand range

  • wave expand cursor

  • wave collapse all

  • wave collapse range

  • wave collapse cursor

These commands have the same behavior as the corresponding menu and toolbar selections. If valid times are not specified, for wave expand range or wave collapse range, no action is taken. These commands affect all Waveform panes in the Wave window to which the command applies.
