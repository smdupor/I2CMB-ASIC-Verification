PrintSVPackageLoadingAttribute

Section [vlog]

This variable prints the attribute placed upon SV packages during package import when true (1). The attribute will be ignored when this variable entry is false (0). The attribute name is “package_load_message.” The value of this attribute is a string literal.

Syntax

PrintSVPackageLoadingAttribute = {0 | 1}

Arguments

The arguments are described as follows:

  • 0 — False

  • 1 — (default) True
