Verilog-XL Compatible Compiler Arguments

The Verilog compiler supports arguments that are equivalent to Verilog-XL arguments, simplifying the porting of a design to Questa SIM.

See the vlog command for a description of each argument.

+define+<macro_name>[=<macro_text>]
+delay_mode_distributed
+delay_mode_path
+delay_mode_unit
+delay_mode_zero
-f <filename>
+incdir+<directory>
+mindelays
+maxdelays
+nowarn<mnemonic>
+typdelays
-u
