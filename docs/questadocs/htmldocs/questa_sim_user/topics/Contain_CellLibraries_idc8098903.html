Cell Libraries

Many ASIC and FPGA vendors’ Verilog cell libraries are compatible with Questa SIM Verilog.

Cell models generally contain Verilog “specify blocks” that describe the path delays and timing constraints for the cells. See Section 14 in the IEEE Std 1364-2005 for details on specify blocks, and Section 15 for details on timing constraints. Questa SIM Verilog fully implements specify blocks and timing constraints as defined in IEEE Std 1364 along with some Verilog-XL compatible extensions.
