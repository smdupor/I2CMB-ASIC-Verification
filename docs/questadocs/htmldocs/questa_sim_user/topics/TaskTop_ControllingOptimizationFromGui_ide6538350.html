Controlling Optimization from the GUI

Optimization (vopt) in the GUI is controlled from the Simulate > Design Optimization dialog box. You can use the Visibility tab restore total design visibility during optimization.

Procedure

  1. From the main menu, choose the following:

    Simulate > Design Optimization

  2. Click the Visibility tab.
  3. Select the following:

    Apply full visibility to all modules (full debug mode)

  4. Click the Design tab and do the following:
    1. Select the top-level design unit to simulate.
    2. Specify an Output Design Name.
    3. Select Start Immediately.
  5. Click OK.
