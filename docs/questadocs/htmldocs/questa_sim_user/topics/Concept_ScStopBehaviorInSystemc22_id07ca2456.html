sc_stop Behavior in SystemC-2.2

When encountered during the simulation run in command line mode, the sc_stop() function stops the current simulation and causes Questa SIM to exit.

In the GUI mode, a dialog box appears asking you to confirm the exit. This is the default operation of sc_stop(). If you want to change the default behavior of sc_stop, you can change the setting of the OnFinish variable in the modelsim.ini file. To change the behavior interactively, use the ‑onfinish argument to the vsim command.
