New Directory Path With $fopen

The $fopen systemtask has been extended to create a new directory path if the path does not currently exist.

You must set the CreateDirForFileAccess modelsim.ini variable to '1' to enable this feature. For example: your current directory contains the directory “dir_1 with no other directories below it and the CreateDirForFileAccess variable is set to “1”. Executing the following line of code:

fileno = $fopen("dir_1/nodir_2/nodir_3/testfile", "w");

creates the directory path nodir_2/nodir_3 and opens the file “testfile” in write mode.
