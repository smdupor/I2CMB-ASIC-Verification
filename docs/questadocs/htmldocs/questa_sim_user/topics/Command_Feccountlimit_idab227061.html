FecCountLimit

Section [vsim]

This variable limits the number of counts that are tracked for Focused Expression Coverage. Specifically, when a bin has reached the specified count, coverage will ignore further tracking of the inputs linked to the bin.

Note:

If you change this value from the default you may affect simulation performance.

Syntax

FecCountLimit = {<n> | 0 }

Arguments

The arguments are described as follows:

  • <n> — Specifies the count limit for FEC. The default is 1.

  • 0 — Specifies an unlimited count
