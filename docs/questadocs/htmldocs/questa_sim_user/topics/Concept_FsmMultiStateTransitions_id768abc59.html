FSM Multi-State Transitions

A multi-state transition is also known as a state sequence.

In the coverage domain, using the switch -fsmmultitrans with vcom or vlog or vopt yields a metric sometimes known as sequence coverage, since it measures the progress of an FSM through a sequence of states. The FSM recognition messages shows multi-state transitions as:

# 		S0 => S1 => S0

When you specify -fsmmultitrans, you can view this information in the:

  • FSM Recognition Info Note in the transcript, specifically the multi-state transition table.

  • FSM Coverage Text Report, specifically the covered transition and uncovered transition tables.

  • Details window.

  • Code Coverage Analysis window for FSM.
