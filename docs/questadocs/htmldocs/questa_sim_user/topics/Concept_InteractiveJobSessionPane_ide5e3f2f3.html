Interactive Job Session Pane

The Interactive Job Session pane provides a command line for interacting with jobs. Commands you enter affect the job currently selected in the Running Jobs portion of the dialog box.

See Table 1 for a list of commands you can enter in the Interactive Job Session pane.

Note:

If you check Advanced Mode, you can enter any Questa SIM command at the prompt. However, you need to be careful as many Questa SIM commands will not function properly with JobSpy.
