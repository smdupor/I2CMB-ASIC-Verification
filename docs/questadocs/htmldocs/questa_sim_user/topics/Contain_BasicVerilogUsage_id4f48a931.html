Basic Verilog Usage

Basic Verilog usage include the steps of compiling, optimizing, loading, and simulating.

Generally you perform the steps in the following order:

  1. Compile your Verilog code into one or more libraries with the vlog command. See Verilog Compilation for details.

  2. (Optional) Elaborate and optimize your design with the vopt command. For more information, refer to Chapter Optimizing Designs with vopt, Optimizing Designs with vopt and Optimization Considerations for Verilog Designs.

  3. Load your design with the vsim command. Refer to Verilog Simulation.

  4. Simulate the loaded design and debug as needed.
