CoverClkOptBuiltins

Section [vcom]

This variable enables clock optimization builtins for code coverage. When these clock optimizations are enabled, some branches of VHDL code may be excluded from code coverage, and given a code of ECOP (when not hit).

Syntax

CoverClkOptBuiltins = {0 | 1}

Arguments

The arguments are described as follows:

  • 0 — Off

  • 1 — (default) On

Description

Refer to“Missing Branches in VHDL and Clock Optimizations” for more details.
