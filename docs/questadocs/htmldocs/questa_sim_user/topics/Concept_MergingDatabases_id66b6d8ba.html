Merging Databases

When merging coverage databases offline using vcover merge, the following parameters must be the same for a given scope:

  • coverage weighting

  • covergroup type name

  • covergroup variable name

  • coverpoint name

  • bin name

If coverage data exists in the source database file but does not match the data in the target database, then it will be created in the target database.
