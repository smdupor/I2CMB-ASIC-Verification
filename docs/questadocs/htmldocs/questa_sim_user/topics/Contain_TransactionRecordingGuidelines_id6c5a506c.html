Transaction Recording Guidelines

Rules and guidelines apply to all transaction recording, regardless of the language in which the transaction is recorded.
Restriction:

The checkpoint/restore commands are not supported for transactions.

For language-specific instructions and deviations from these general truths, see: Recording Transactions in Verilog and VHDL

For SystemC Verification (SCV) specific limitations and implementation details, see: SCV Limitations
