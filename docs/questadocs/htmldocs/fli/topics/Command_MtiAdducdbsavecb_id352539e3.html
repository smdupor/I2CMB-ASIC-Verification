mti_AddUCDBSaveCB()

Adds the specified function to the coverage save callback list of the simulator.

Syntax

mti_AddUCDBSaveCB ( region, func, param )

Arguments

Name

Type

Description

region

mtiRegionIdT

Valid handle for the region on which the function will be called.

func

mtiUCDBSaveFuncPtrT

Function pointer which will be called on the specified region.

param

void *

The parameter to the function. Can also be null.

Return Values

Nothing

Description

The Coverage Save command will call this function on the specified region.
