Locating your Hardware Identifier on Linux Platforms

On Linux platforms you will need your workstation ID to obtain a license.

Procedure

Determining your hardware identifier.
Table 1. Determining Linux Workstation ID

Platform

Syntax

Notes

Linux

/sbin/ifconfig eth0

Look for the line that reads something like:

“Ethernet HWaddr 00:00:00:00:00:00.”

Remove the colons and you have the required 12‑digit ID.
