Design Library Regeneration

When you upgrade between calendar releases (for example, from 2019.1 to 2020.1), you must regenerate your design libraries. This is not true for quarterly releases of the same calendar year, such as 2019.1 to 2019.2.

Libraries are compatible between different quarterly releases of the product, as long as the version number is the same. For example, 2019.1 libraries are compatible with 2019.2, but not with older releases.
