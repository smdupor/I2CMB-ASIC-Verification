Installing SystemC Compilers on Windows Platforms

For the Windows platform, you must install the compilers manually. The following steps describe how to obtain and install your required compiler.

Procedure

  1. Install the product.

    This installs the base product and the documentation.

  2. Navigate to the following site.

    https://support.mentor.com

  3. Locate the appropriate MinGW ZIP file:

    • questasim_gcc-4.2.1-mingw32vc12.zip

    • questasim_gcc-4.5.0-mingwv64vc12.zip for Windows 64-bit

  4. Extract the contents of the ZIP file to the top level of your install directory.
