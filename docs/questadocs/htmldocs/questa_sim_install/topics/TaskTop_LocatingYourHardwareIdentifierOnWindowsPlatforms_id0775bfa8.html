Locating Your Hardware Identifier on Windows Platforms

On Windows platforms, the hardware identifier may be either a hardware security key or an Ethernet ID.

Procedure

Locate your hardware identifier in one of two ways.

  • Locate your hardware identifier.

    • Security key ID number — check the number printed on the key.

    • Ethernet ID —

      1. select Start > Run

      2. Open “cmd” to open a command prompt.

      3. At the command prompt type:

        ipconfig -all | more

      4. Check the configuration listing for your Ethernet “Physical Address”.
