Design Object Icons and Their Meanings

The color and shape of icons convey information about the language and type of a design object.

Table 1 shows the icon colors and the languages they indicate.

Table 1. Design Object Icons

Icon color

Design Language

light blue

dark blue

green

orange

virtual object
