Source Toolbar

The Source toolbar allows you to perform several activities on Source windows.
Figure 1. Source Toolbar

Table 1. Source Toolbar Buttons

Button

Name

Shortcuts

Description

Previous Zero Hits

None

Jump to previous line with zero coverage.

Next Zero Hits

None

Jump to next line with zero coverage.

Source Annotation

Menu: Source > Show Annotation

Allows Debugging with Source Annotation in every open source file.

Clear Bookmarks

Menu: Source > Clear Bookmarks

Removes any bookmarks in the active source file.
