Compile Toolbar Tab

The Compile Toolbar Tab provides access to compile actions.
Figure 1. Compile Tab

Table 1. Compile Toolbar Tab Buttons

Button

Name

Shortcuts

Description

Compile

Command: vcom or vlog

Menu: Compile > Compile

Opens the Compile Source Files dialog box.

Compile All

Command: vcom or vlog

Menu: Compile > Compile All

Compiles all files in the open project.

Simulate

Command: vsim

Menu: Simulate > Start Simulation

Opens the Start Simulation dialog box.

Break

Menu: Simulate > Break

Hotkey: Break

Stop a compilation, elaboration, or the current simulation run.

Compile Out of Date

  

Recompile if changes have been made to any source files.
