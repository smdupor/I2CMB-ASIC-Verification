Add Bookmark

Command (Wave window only): bookmark add wave

Menu (Wave window only): Add > To > Wave > Bookmark

Clicking this button bookmarks the current view of the active window.

Click and hold the button to open a drop down menu with the following options:

  • Add Current View

  • Add Custom ...

  • Set Default Action

Delete All Bookmarks

Command (Wave window only): bookmark delete wave -all

Removes all bookmarks, after prompting for your confirmation.

Click and hold the button to open a drop down menu with the following options:

  • Active Window

  • All Windows
