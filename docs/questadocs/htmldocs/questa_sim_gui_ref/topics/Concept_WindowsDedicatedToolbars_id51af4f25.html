Windows With Dedicated Toolbars

Buttons that function only in a specific window are available in a bar at the top of the window.

Click the Show/Hide Toolbar button in the window title bar to display the window specific buttons (Figure 1). You can add buttons to any currently open window. Refer to Customizing Button and Tab Location for more information.

Figure 1. Window Specific Buttons

The following windows have default buttons or widgets:

  • ATV Window

  • FSM Viewer Window

  • Memory Data Window

  • Objects Window

  • Processes Window

  • Source Window

  • Verification Results Analysis Window
