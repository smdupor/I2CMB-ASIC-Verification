Panning the View of the Dataflow Window

You can pan the view of the Dataflow window with the mouse or keyboard.

Procedure

Pan the view of the Dataflow window using one of the following methods:

If you want to...

Do the following...

Pan with the Mouse

In Zoom mode, pan with the middle mouse button. In Pan mode, pan with the left mouse button. In Select mode, pan with the Ctrl key and the middle mouse button.

Pan with the Keyboard

Use the arrow keys to pan the view. Shift+<arrow key> pans to the far edge of the view. Ctrl+<arrow key> pans by a moderate amount.
