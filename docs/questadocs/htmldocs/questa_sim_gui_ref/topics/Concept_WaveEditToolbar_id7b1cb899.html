Wave Edit Toolbar

The Wave Edit toolbar provides easy access to tools for modifying an editable wave.
Figure 1. Wave Edit Toolbar

Table 1. Wave Edit Toolbar Buttons

Button

Name

Shortcuts

Description

Insert Pulse

Menu: Wave > Wave Editor > Insert Pulse

Command: wave edit insert_pulse

Insert a transition at the selected time.

Delete Edge

Menu: Wave > Wave Editor > Delete Edge

Command: wave edit delete

Delete the selected transition.

Invert

Menu: Wave > Wave Editor > Invert

Command: wave edit invert

Invert the selected section of the waveform.

Mirror

Menu: Wave > Wave Editor > Mirror

Command: wave edit mirror

Mirror the selected section of the waveform.

Change Value

Menu: Wave > Wave Editor > Value

Command: wave edit change_value

Change the value of the selected section of the waveform.

Stretch Edge

Menu: Wave > Wave Editor > Stretch Edge

Command: wave edit stretch

Move the selected edge by increasing/decreasing waveform duration.

Move Edge

Menu: Wave > Wave Editor > Move Edge

Command: wave edit move

Move the selected edge without increasing/decreasing waveform duration.

Extend All Waves

Menu: Wave > Wave Editor > Extend All Waves

Command: wave edit extend

Increase the duration of all editable waves.
