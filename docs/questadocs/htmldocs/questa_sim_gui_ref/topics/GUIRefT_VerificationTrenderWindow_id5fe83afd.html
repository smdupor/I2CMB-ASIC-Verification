Verification Trender Window

To access:

  • Highlight one or more UCDBs

  • Trend Analysis > View Trender

  • view trender command

The Verification Trender window is used for analyzing coverage trends in verification. This window has a feature for user-customizable column headings. It requires a “qvman” license to view.

Description

Figure 1. Trender Window

The columns that appear in the Trender window are the object name, the type of object and the dates when the data was saved. Right-click in the column headings to display a list of all column layout and configuration options. You can these options to create custom column layouts which will preserve the columns you commonly use.
