Add Menu

The Add menu provides access to options for populating windows with data from other windows.
Table 1. Add Menu — Item Description

Menu Item

Description

To Wave

Add information to the Wave window

To List

Add information to the List window

To Log

Add information to the Log file

To Schematic

Add information to the Schematic window

To Dataflow

Add information to the Dataflow window

Window Pane

Add an additional pane to the Wave window. You can remove this pane by selecting Wave > Delete Window Pane.
