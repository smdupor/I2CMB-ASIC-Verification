Analysis Toolbar

The Analysis (coverage) toolbar allows you to control aspects of the Code Coverage Analysis window.

A dropdown box that allows you to specify the type of code coverage to view in the Code Coverage Analysis Window.

Menu: N/A

All covered (hit) items are displayed.

Missed

Menu: N/A

All missed items (not executed) are displayed.

Excluded
