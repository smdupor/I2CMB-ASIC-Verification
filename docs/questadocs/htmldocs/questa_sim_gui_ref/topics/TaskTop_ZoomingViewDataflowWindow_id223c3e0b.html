Zooming the View of the Dataflow Window

Several zoom controls are available for changing the view of the Dataflow window, including mouse strokes, toolbar icons and a mouse scroll wheel.

Procedure

You can zoom the view of the Dataflow window using one of the following methods:

If you want to...

Do the following...

Zoom Full — Fills the Dataflow window with all visible data

Mouse stroke

Up/Left. Middle mouse button in Select and Pan mode, Left mouse button in Zoom mode.

Menu

Dataflow > Zoom Full

Zoom Toolbar

Zoom Full

Zoom Out

Mouse stroke

Up/Right. Middle mouse button in Select and Pan mode, Left mouse button in Zoom mode.

Menu

Dataflow > Zoom Out

Zoom Toolbar

Zoom Out

Mouse Scroll

Push forward on the scroll wheel.

Zoom In

Menu

DataFlow > Zoom In

Zoom Toolbar

Zoom In

Mouse Scroll

Pull back on the scroll wheel.

Zoom Area — Fills the Dataflow window with the data within the bounding box.

Mouse stroke

Down/Right

Mouse stroke

Down/Left
