For this exercise, you run all simulations from the PA_tutorial directory.

Script Files

The Questa/scripts/ directory contains the following do files for compiling all simulations:

  • compile_rtl.do — Compiles RTL source files.

  • compile_gls.do — Compiles GLS source files for the Liberty simulation.

The Questa/scripts/tkgui/ directory contains the following do files for running all simulations:

  • analyze_config1.do — Processes UPF files for the first Power Aware RTL simulation.

  • analyze_config2.do — Processes UPF files for the second Power Aware RTL simulation.

  • analyze_imp.do — Processes UPF files for the third Power Aware RTL simulation.

  • analyze_gls.do — Processes UPF and Liberty files for Power Aware gate-level simulation (GLS).

  • liberty_gls.do — Reads Liberty files and creates Liberty attribute library.

  • doit_rtl.do — Configures the first two RTL Power Aware simulations, invokes the vsim command, and runs sim.do.

  • doit_imp.do — Configures the third RTL and Liberty (GLS) Power Aware simulations, invokes the vsim command, and runs sim.do.

  • sim.do — In VSIM mode, sets up the Wave window display, specifies Power Aware messaging, and controls the RTL and GLS Power Aware simulations to completion.
