Run the Simulation

You will now run the simulation and view the profiling data.

Procedure

  1. Enable the statistical sampling profiler.
    1. Select Tools > Profile > Performance or click the Performance Profiling icon in the toolbar.

      This must be done prior to running the simulation. Questa SIM is now ready to collect performance data when the simulation is run.

  2. Run the simulation.
    1. Type “run 5 ms” at the VSIM> prompt.

      Notice that the number of samples taken is displayed both in the Transcript and the Main window status bar (Figure 1). (Your results may not match those in the figure.) Also, Questa SIM reports the percentage of samples that were taken in your design code (versus in internal simulator code).
