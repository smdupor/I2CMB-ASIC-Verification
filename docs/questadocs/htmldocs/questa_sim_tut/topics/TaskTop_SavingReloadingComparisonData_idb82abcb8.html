Saving and Reloading Comparison Data

You can save comparison data for later viewing, either in a text file or in files that can be reloaded into Questa SIM.

To save comparison data so it can be reloaded into Questa SIM, you must save two files. First, you save the computed differences to one file; next, you save the comparison configuration rules to a separate file. When you reload the data, you must have the reference dataset open.

Procedure

  1. Save the comparison data to a text file.
    1. In the Main window, select Tools > Waveform Compare > Differences > Write Report.
    2. Click Save.

      This saves compare.txt to the current directory.

    3. Type notepad compare.txt at the VSIM> prompt to display the report (Figure 1).
      Figure 1. Coverage data saved to a text file

    4. Close Notepad when you have finished viewing the report.
  2. Save the comparison data in files that can be reloaded into Questa SIM.
    1. Select Tools > Waveform Compare > Difference > Save.
    2. Click Save.

      This saves compare.dif to the current directory.

    3. Select Tools > Waveform Compare > Rules > Save.
    4. Click Save.

      This saves compare.rul to the current directory.

    5. Select Tools > Waveform Compare > End Comparison.
  3. Reload the comparison data.
    1. With the Structure (sim) window active, select File > Open.
    2. Change the Files of Type to Log Files (*.wlf) (Figure 2).
      Figure 2. Displaying Log Files in the Open Dialog Box

    3. Double-click gold.wlf to open the dataset.
    4. Select Tools > Waveform Compare > Reload.

      Since you saved the data using default file names, the dialog box should already have the correct Waveform Rules and Waveform Difference files specified (Figure 3).

      Figure 3. Reloading Saved Comparison Data

    5. Click OK.

      The comparison reloads. You can drag the comparison object to the Wave or List window to view the differences again.
