Initialize a Memory

In Questa SIM, it is possible to initialize a memory using one of three methods: from an exported memory file, from a fill pattern, or from both.

First, you will initialize a memory from a file only. You will use the one you exported previously, data_mem.mem.

Procedure

  1. View instance /ram_tb/spram3/mem.
    1. Double-click the /ram_tb/spram3/mem instance in the Memory List window.

      This will open a new Memory Data window to display the contents of /ram_tb/spram3/mem. Familiarize yourself with the contents so you can identify changes once the initialization is complete.

    2. Right-click and select Properties to bring up the Properties dialog box.
    3. Change the Address Radix to Decimal, Data Radix to Binary, Words per Line to “1” and click OK.
  2. Initialize spram3 from a file.
    1. Right-click anywhere in the data column and select Import Data Patterns to bring up the Import Memory dialog box (Figure 1).
      Figure 1. Import Memory Dialog Box

      The default Load Type is File Only.

    2. Type “data_mem.mem” in the Filename field.
    3. Click OK.

      The addresses in instance /ram_tb/spram3/mem are updated with the data from data_mem.mem (Figure 2).

      Figure 2. Initialized Memory from File and Fill Pattern

    In this next step, you will experiment with importing from both a file and a fill pattern. You will initialize spram3 with the 250 addresses of data you exported previously into the relocatable file reloc.mem. You will also initialize 50 additional address entries with a fill pattern.

  3. Import the /ram_tb/spram3/mem instance with a relocatable memory pattern (reloc.mem) and a fill pattern.
    1. Right-click in the data column of spram3 and select Import Data Patterns to bring up the Import Memory dialog box.
    2. For Load Type, select Both File and Data.
    3. For Address Range, select Addresses and enter “0” as the Start address and “300” as the End address.

      This means that you will be loading the file from 0 to 300. However, the reloc.mem file contains only 251 addresses of data. Addresses 251 to 300 will be loaded with the fill data you specify next.

    4. For File Load, select the MTI File Format and enter reloc.mem in the Filename field.
    5. For Data Load, select a Fill Type of Increment.
    6. In the Fill Data field, set the seed value of “0” for the incrementing data.
    7. Click OK.
    8. View the data near address 250 by double-clicking on any address in the Address column and entering “250.”

    You can see the specified range of addresses overwritten with the new data. Also, you can see the incrementing data beginning at address 251 (Figure 3).

    Figure 3. Data Increments Starting at Address 251

    Now, before you leave this section, go ahead and clear the memory instances already being viewed.

  4. Right-click in one of the Memory Data windows and select Close All.
