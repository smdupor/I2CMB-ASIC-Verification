Interactive Debugging Commands

The Memory Data windows can also be used interactively for a variety of debugging purposes. The features described in this section are useful for this purpose.

Procedure

  1. Open a memory instance and change its display characteristics.
    1. Double-click instance /ram_tb/dpram1/mem in the Memory List window.
    2. Right-click in the dpram1 Memory Data window and select Properties.
    3. Change the Address and Data Radix to Hexadecimal.
    4. Select Words per line and enter “2.”
    5. Click OK. The result should be as shown in Figure 1.
      Figure 1. Original Memory Content

  2. Initialize a range of memory addresses from a fill pattern.
    1. Right-click in the data column of /ram_tb/dpram1/mem and select Change to open the Change Memory dialog box (Figure 2).
      Figure 2. Changing Memory Content for a Range of Addresses

    2. Select Addresses and enter the start address as “0x00000006” and the end address as “0x00000009.” The "0x" hex notation is optional.
    3. Select Random as the Fill Type.
    4. Enter “0” as the Fill Data, setting the seed for the Random pattern.
    5. Click OK.

      The data in the specified range are replaced with a generated random fill pattern (Figure 3).

      Figure 3. Random Content Generated for a Range of Addresses

  3. Change contents by highlighting.

    You can also change data by highlighting them in the Address Data pane.

    1. Highlight the data for the addresses 0x0000000c:0x0000000e, as shown in Figure 4.
      Figure 4. Changing Memory Contents by Highlighting

    2. Right-click the highlighted data and select Change.

      This brings up the Change memory dialog box. Note that the Addresses field is already populated with the range you highlighted.

    3. Select Value as the Fill Type. (Refer to Figure 5)
    4. Enter the data values into the Fill Data field as follows: 31 32 33 34.
      Figure 5. Entering Data to Change

    5. Click OK.

      The data in the address locations change to the values you entered (Figure 6).

      Figure 6. Changed Memory Contents for the Specified Addresses

  4. Edit data in place.

    To edit only one value at a time, do the following:

    1. Double-click any value in the Data column.
    2. Enter the desired value and press the Enter or Return key on your keyboard.

      If you needed to cancel the edit function, press the Esc key on your keyboard.
