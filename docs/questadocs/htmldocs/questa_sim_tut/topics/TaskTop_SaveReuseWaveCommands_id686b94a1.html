Save and Reuse the Wave Commands

You can save the commands that Questa SIM used to create the waveforms. You can load this “format” file at a later time to re-create the waves. In this exercise, we will save the commands, quit and reload the simulation, and then open the format file.

Procedure

  1. Save the wave commands to a format file.
    1. Select File > Save Format in the menu bar to open the Save Format dialog box.
      Figure 1. Save Format Dialog Box

      The default file name is wave.do.

    2. Click the OK button to save a DO file named wave.do to the current directory.
    3. Close the Wave window by clicking Close icon (x) in the top right corner or by selecting View > Wave in the menus.
  2. Quit and then reload the design.
    1. In the Main window, select Simulate > End Simulation, and click Yes to confirm you want to quit simulating.
    2. Enter the following command at the Questa SIM> prompt.

      vsim counter

  3. Open the format file.
    1. Select View > Wave to open the Wave window.
    2. Select File > Load > Macro File from the menu bar.
    3. Double-click wave.do to open the file.

      The waves you created earlier in the lesson reappear. If waves do not appear, you probably did not load the counter design unit.
