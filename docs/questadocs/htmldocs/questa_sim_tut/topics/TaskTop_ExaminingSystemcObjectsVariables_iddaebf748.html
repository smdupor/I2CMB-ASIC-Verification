Examining SystemC Objects and Variables

To examine the value of a SystemC object or variable, you can use the examine command or view the value in the Objects window.

Procedure

  1. View the value and type of an sc_signal.
    1. Enter the “show” command at the CDBG > prompt to display a list of all design objects, including their types, in the Transcript.

      In this list, you will see that the type for dataerror is “boolean” (sc_logic for VHDL) and counter is “int” (Figure 1).

      Figure 1. Output of show Command

    2. Enter the “examine dataerror” command at the CDBG > prompt.

      The value returned is “true.”

  2. View the value of a SystemC variable.
    1. Enter the “examine counter” command at the CDBG > prompt to view the value of this variable.

      The value returned is “32'hFFFFFFFF”.
