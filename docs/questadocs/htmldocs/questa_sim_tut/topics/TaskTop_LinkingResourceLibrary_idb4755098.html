Linking to the Resource Library

Linking to a resource library requires that you specify a "search library" when you invoke the simulator.

Procedure

Specify a search library during simulation.
  1. Click the Simulate icon on the Main window toolbar.
  2. Click the ’+’ icon next to the work library and select test_counter.
  3. Click the Libraries tab.
  4. Click the Add button next to the Search Libraries field and browse to parts_lib in the resource_library directory you created earlier in the lesson.
  5. Click OK.

    The dialog box should have parts_lib listed in the Search Libraries field (Figure 1).

  6. Click OK.

    The design loads without errors.
