Creating the Project

Now you will create a project that contains tcounter.v, the counter’s test bench.

Procedure

  1. Create the project.
    1. Select File > New > Project.
    2. Type counter in the Project Name field.
    3. Do not change the Project Location field or the Default Library Name field. (The default library name is work.)
    4. Make sure “Copy Library Mappings” is selected. The default modelsim.ini file will be used.
    5. Click OK.
  2. Add the test bench to the project.
    1. Click Add Existing File in the Add items to the Project dialog box.
    2. Click the Browse button and select tcounter.v in the “Select files to add to project” dialog box.
    3. Click Open.
    4. Click OK.
    5. Click Close to dismiss the “Add items to the Project” dialog box.

      The tcounter.v file is listed in the Project window.

  3. Compile the test bench.
    1. Right-click tcounter.v and select Compile > Compile Selected.
