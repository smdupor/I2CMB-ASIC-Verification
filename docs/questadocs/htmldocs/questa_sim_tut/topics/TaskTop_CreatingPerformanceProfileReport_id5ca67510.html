Creating a Performance Profile Report

Questa SIM allows you to create different profile reports based on the profiler data.

Procedure

Create a call tree type report of the performance profile.
  1. With the Calltree window open, select Tools > Profile > Profile Report from the menus to open the Profile Report dialog box.
  2. In the Profile Report dialog box (Figure 1), select the Call Tree Type.
    Figure 1. The Profile Report Dialog Box

  3. In the Performance/Memory data section select Performance only.
  4. Specify the Cutoff percent as 3%.
  5. Select Write to file and type calltree.rpt in the file name field.
  6. View file is selected by default when you select Write to file. Leave it selected.
  7. Click OK.

    The calltree.rpt report file will open automatically in Notepad (Figure 2).

    Figure 2. The calltree.rpt Report

    You can also output this report from the command line using the profile report command. See the Questa SIM Command Reference for details.
