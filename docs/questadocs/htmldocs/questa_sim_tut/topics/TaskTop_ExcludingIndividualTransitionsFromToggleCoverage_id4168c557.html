Excluding Individual Transitions from Toggle Coverage

Questa SIM allows you to exclude individual logic transitions from toggle coverage statistics.

Procedure

  1. Open the Code Coverage Analysis and Coverage Details windows.
    1. Select View > Coverage > Code Coverage Analysis to open the Coverage Analysis window.
    2. Select View > Coverage > Details to open the Coverage Details window.
  2. Select the Toggle coverage type in the toolbar of the Code Coverage Analysis window.
    Figure 1. Select Toggle Coverage Type

    With test_sm selected in the Structure (sim) window, the Code Coverage Analysis now displays all toggles, by instance, for test_sm.

  3. Select the addr signal in the Code Coverage Analysis window to display all logic transitions for that signal in the Coverage Details window.
  4. Exclude the 1H->Z transition from toggle coverage statistics the next time the simulation is run.
    1. Right-click the 1H->Z transition.
    2. Select Exclude transition from the popup menu. (Figure 2)
      Figure 2. Select Exclude transition

      And ‘E’ will appear next to the transition to indicate that it is now excluded.

  5. Unexclude the 1H->Z transition you just excluded.
    1. Right-click the 1H->Z transition.
    2. Select Unexclude transition from the popup menu.
