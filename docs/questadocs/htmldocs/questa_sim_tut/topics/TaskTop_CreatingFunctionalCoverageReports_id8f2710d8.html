Creating Functional Coverage Reports

You can create functional coverage reports using dialog boxes accessible through the GUI or via commands entered at the command line prompt.

Procedure

  1. Create a functional coverage report using the GUI.
    1. Right-click in the Cover Directives window and select Report. This opens the Functional coverage report dialog box (Figure 1).
      Figure 1. Functional Coverage Report Dialog Box

    2. With All coverage items selected, select Covergroups only.
    3. Select Include covergroups options.
    4. Select OK to write the report to the file fcover_report.txt.

      The actions taken in the GUI are echoed in the transcript as follows:

      coverage report -detail -cvg -comments -option 
					-output fcover_report.txt /.

      The report will appear automatically in Questa SIM Notepad as shown in Figure 2.

      Figure 2. Functional Coverage Text Report

  2. You can also create textual, html, and exclusion coverage reports using the Tools > Coverage Report menu selection.
