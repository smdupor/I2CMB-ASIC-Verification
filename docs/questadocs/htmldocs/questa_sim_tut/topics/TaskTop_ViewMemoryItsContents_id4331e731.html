View a Memory and Its Contents

The Memory List window lists all memory instances in the design, showing for each instance the range, depth, and width. Double-clicking an instance opens a window displaying the memory data.

Procedure

  1. Open the Memory List window and view the data of a memory instance.
    1. If the Memory List window is not already open, select View > Memory List.

      A Memory List window is shown in Figure 1.

      Figure 1. The Memory List Window

    2. Double-click the /ram_tb/spram1/mem instance in the memory list to view its contents.

      A Memory Data window opens displaying the contents of spram1. The first column (blue hex characters) lists the addresses, and the remaining columns show the data values.

      If you are using the Verilog example design, the data is all X (Figure 2) because you have not yet simulated the design.

      Figure 2. Verilog Memory Data Window

      If you are using the VHDL example design, the data is all zeros (Figure 3).

      Figure 3. VHDL Memory Data Window

    3. Double-click the instance /ram_tb/spram2/mem in the Memory List window. This opens a second Memory Data window that contains the addresses and data for the spram2 instance. For each memory instance that you click in the Memory List window, a new Memory Data window opens.
  2. Simulate the design.
    1. Click the Run -All icon in the Main window.

      A Source window opens showing the source code for the ram_tb file at the point where the simulation stopped.

    VHDL: In the Transcript window, you will see NUMERIC_STD warnings that can be ignored and an assertion failure that is functioning to stop the simulation. The simulation itself has not failed.

    1. Click the Memory ...spram1/mem tab to bring that Memory data window to the foreground. The Verilog data fields are shown in Figure 4.
      Figure 4. Verilog Data After Running Simulation

      The VHDL data fields are shown in Figure 5.

      Figure 5. VHDL Data After Running Simulation

  3. Change the address radix and the number of words per line for instance /ram_tb/spram1/mem.
    1. Right-click anywhere in the spram1 Memory Data window and select Properties.
    2. The Properties dialog box opens (Figure 6).
      Figure 6. Changing the Address Radix

    3. For the Address Radix, select Decimal. This changes the radix for the addresses only.
    4. Change the Data Radix to Symbolic.
    5. Select Words per line and type â1â in the field.
    6. Click OK.

    You can see the Verilog results of the settings in Figure 7 and the VHDL results in Figure 8.
