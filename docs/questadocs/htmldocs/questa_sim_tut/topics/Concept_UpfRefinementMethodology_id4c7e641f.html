UPF Refinement Methodology

This tutorial utilizes the UPF 2.0 Successive Refinement Methodology for power intent specification.

This methodology (illustrated in Figure 1) is useful for the development of a power intent specification for IP-based systems and reflects the realities of SoC designs today. To illustrate this methodology, the tutorial involves running three different power aware simulations on the same RTL design. The second and third simulations each involve processing one additional UPF file to reflect the updated power intent information added from the previous simulation. Finally, the tutorial includes a fourth simulation to illustrate the use of Liberty files in Questa SIM.
