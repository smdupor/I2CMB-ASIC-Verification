Creating the Test Dataset

The test dataset is the .wlf file that will be compared against the reference dataset. Like the reference dataset, the test dataset can be a saved dataset, the current simulation dataset, or any part of the current simulation dataset.

To simplify matters, you will create the test dataset from the simulation you just ran. However, you will edit the test bench to create differences between the two runs.

Procedure

  1. Verilog
  2. Edit the test bench.
    1. Select File > Open and open test_sm.v.
    2. Right-click in the Source window for the test_sm.v file and uncheck the Read Only selection in the popup menu.
    3. Scroll to line 122, which looks like this:
      @ (posedge clk) wt_wd('h10,'haa);
    4. Change the data pattern ‘haa’ to ‘hab’:
      @ (posedge clk) wt_wd('h10,'hab);
    5. Select File > Save to save the file.
  3. Compile the revised file and rerun the simulation.
    vlog test_sm.v
vopt +acc test_sm -o opt_test_sm -noincr
vsim opt_test_sm
add wave *
run 750 ns
  4. VHDL
  5. Edit the test bench.
    1. Select File > Open and open test_sm.vhd.
    2. Scroll to line 151, which looks like this:
      wt_wd ( 16#10#, 16#aa#, clk, into );
    3. Change the data pattern ‘aa’ to ‘ab’:
      wt_wd ( 16#10#, 16#ab#, clk, into );
    4. Select File > Save to save the file.
  6. Compile the revised file and rerun the simulation.

    • VHDL

      vcom test_sm.vhd
vopt +acc test_sm -o opt_test_sm
vsim opt_test_sm
add wave *
run 750 ns
