Saving and Reusing the Window Format

If you close the Wave window, the tool discards any configurations you made to the window (for example, signals added, cursors set). However, you can use the Save Format command to capture the current Wave window display and signal preferences to a .do file. You open the .do file later to recreate the Wave window as it appeared when the file was created.

Format files are design-specific; use them only with the design you were simulating when they were created.

Procedure

  1. Save a format file.
    1. In the Wave window, select File > Save Format.
    2. In the Pathname field of the Save Format dialog box, leave the file name set to wave.do and click OK.
    3. Close the Wave window.
  2. Load a format file.
    1. In the Main window, select View > Wave.
    2. Undock the window.

      All signals and cursor(s) that you had set are gone.

    3. In the Wave window, select File > Load.
    4. In the Open Format dialog box, select wave.do and click Open.

      Questa SIM restores the window to its previous state.

    5. Close the Wave window when you are finished by selecting File > Close Window.
