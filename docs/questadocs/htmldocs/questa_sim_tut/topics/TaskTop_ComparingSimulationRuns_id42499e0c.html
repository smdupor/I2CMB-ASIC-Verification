Comparing the Simulation Runs

Questa SIM includes a Comparison Wizard that walks you through the process. You can also configure the comparison manually with menu or command line commands.

Procedure

Create a comparison using the Comparison Wizard.
  1. Select Tools > Waveform Compare > Comparison Wizard.
  2. Click the Browse button and select gold.wlf as the reference dataset (Figure 1). Recall that gold.wlf is from the first simulation run.
    Figure 1. First Dialog Box of the Waveform Comparison Wizard

  3. Leaving the test dataset set to Use Current Simulation, click Next.
  4. Select Compare All Signals in the second dialog box (Figure 2) and click Next.
    Figure 2. Second Dialog Box of the Waveform Comparison Wizard

  5. In the next three dialog boxes, click Next, Compute Differences Now, and Finish, respectively.

    Questa SIM performs the comparison and displays the compared signals in the Wave window.
