Lesson Wrap-Up

There is certainly much more involved with passing data back and forth across the boundary between C and Verilog using DPI. What about user-defined types? What about arrays? Structs? 64-bit integers? This particular subject can get into some intricate detail, and we have already covered quite a bit in this lesson towards the goal of helping you understand the basics of passing data through the interface. Most importantly, what you have learned should give you an understanding of how to make use of the DPI header file that the compiler creates in order to make sure your C code is written properly to interface with SystemVerilog.
