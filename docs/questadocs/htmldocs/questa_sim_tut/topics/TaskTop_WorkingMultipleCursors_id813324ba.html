Working with Multiple Cursors

Even more information is available when working with multiple cursors.

Procedure

  1. Add a second cursor.
    1. Click the Insert Cursor icon on the Wave window toolbar.
    2. Right-click the name of the new cursor and delete the text.
    3. Type “B” and press Enter.
    4. Drag cursor B and watch the interval measurement change dynamically (Figure 1).
      Figure 1. Interval Measurement Between Two Cursors

  2. Lock cursor B.
    1. Right-click the yellow time indicator box associated with cursor B (at 56 ns).
    2. Select Lock B from the popup menu.

      The cursor color changes to red and you can no longer drag the cursor (Figure 2).

      Figure 2. A Locked Cursor in the Wave Window

  3. Delete cursor B.
    1. Right-click cursor B (the red box at 56 ns) and select Delete B.
