Rules Usage

In order to use Rules, you must:

  1. Specify a particular rule as the Link of your testplan, just as you would any other coverage object or test data record. For example, if your plan is an Excel based plan, you enter the rule into the Link column of the testplan, using the correct syntax for that specific rule. See “Currently Supported Rules”for rules and syntax

  2. Specify the Type as “Rule”, to indicate that the corresponding item defined in the Link is a rule. Like other coverage objects, a testplan may link to one or more rules.
