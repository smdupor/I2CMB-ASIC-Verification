racedetect off

Disable reporting of races.

Syntax

racedetect off [-fullreport] [-limit <number>]

Arguments

-fullreport

Disables reporting in the racedetect.txt file. The unique report file ("racedetect.unique.txt") continues to be updated.

-limit <number>

Disables reporting after <number> of races have been reported in the "racedetect.txt" file. The default is 10,000.

Description

Disables race detection.
