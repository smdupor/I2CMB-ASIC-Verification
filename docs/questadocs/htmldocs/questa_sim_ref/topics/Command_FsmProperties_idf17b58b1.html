fsm properties

Returns information about the specified finite state machine. This information matches that found in the FSM Properties dialog box.

Syntax

fsm properties <fsm_instance>

Arguments

<fsm_instance>

(required) You can produce a list of available fsm instances with the fsm list command.

Examples

fsm properties /top/A/I0/present_state

Returns: 

top/A/I0/clk {RESET 0 S0 1 S1 2 S2 3 S3 4 S4 5 S5 6 S6 7 S7 8 S8 9} RESET {top/A/I0/#ALWAYS#53 top/A/I0/#ALWAYS#60} {top/A/I0/clk top/A/I0/rst top/A/I0/next_state top/A/I0/enable} {top/A/I0/present_state top/A/I0/en2 top/A/I0/en1 top/A/I0/en0 top/A/I0/dval top/A/I0/fifo_rd}

which follows the form: 

<clk> {<state_encoding>} <reset_state> {process_list} {<clk> <inputs> <outputs>}
