To add a new regression suite, click the Create button, enter a unique name into the dialog box, select a template, and click the Create Suite button. Note: You cannot name a suite global To delete a regression suite, click the trash icon on that row. To generate the files for a regression suite, click the Generate button.

Clicking the name of any regression suite in the table will load the data-entry menu for that suite. The data-entry menu will appear in the navigation pane on the left of the screen.
